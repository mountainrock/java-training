In this example:

We use the navbar class to create a navigation bar.
The navbar-expand-lg class ensures that the navigation bar is expanded on large screens and collapses on smaller screens.
The navbar-light bg-light classes provide a light background for the navigation bar.
The navigation bar contains a logo, a toggle button for mobile navigation, and a collapsible menu.
The navbar-brand class styles the logo.
The navbar-toggler class creates the toggle button.
The collapse navbar-collapse class is used to collapse the navigation menu on smaller screens.
Navigation links are styled using the navbar-nav class.
